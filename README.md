# Mononton (React Native)

## API Documentation
```
https://www.themoviedb.org
```

## Intallation

- Git Clone
```
$ git clone https://github.com/MuhamatZaenalMahmut/mononton.git
```

- Yarn Install
```
$ yarn install
```

- Running in Android
```
$ react-native run-android
```
